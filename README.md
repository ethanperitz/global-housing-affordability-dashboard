# Global Housing Affordability Dashboard

An interactive Streamlit dashboard exploring housing affordability across global cities.

### Live App Link
https://global-housing-affordability-dashboard-ethanperitz.streamlit.app/

## Features
- Compare one or two countries at a time
- Analyze rent burden as a percent of monthly salary
- Compare the number of months of income needed to afford one square meter of housing
- Explore rankings and affordability insights

## Metrics
- **Rent Burden**: Average percent of monthly salary needed to afford a 1-bedroom apartment outside the city center
- **Months per m²**: Approximate number of months of salary required to afford one square meter of housing outside the city center

## Built With
- Python
- Pandas
- Plotly
- Streamlit
